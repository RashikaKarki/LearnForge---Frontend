
# Learnforge Design Guidelines

## 1. Visual Identity

* **Primary Color Palette:** Deep navy (#0B132B), Sky blue (#5BC0BE), and White (#FFFFFF)
* **Accent Colors:** Coral (#F25F5C) and Soft Gray (#EAEAEA)
* **Usage Rule:** Use the primary colors for most UI components. Coral is reserved for call-to-action buttons or highlights.

## 2. Typography

* **Primary Font:** Inter (for headings and body)
* **Secondary Font:** Roboto Mono (for code snippets or technical sections)
* **Font Weights:** 400 (Regular), 600 (Semi-bold), 700 (Bold)
* **Hierarchy:**

  * H1: 36px, Bold
  * H2: 28px, Semi-bold
  * H3: 22px, Semi-bold
  * Body: 16px, Regular
  * Caption: 14px, Regular

## 3. Iconography & Illustration

* **Style:** Simple, flat-line icons with consistent stroke width (2px)
* **Theme:** Modern and minimal, with a focus on clarity and learning
